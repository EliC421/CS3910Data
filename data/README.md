# Data Directory

This directory contains the CSV files and processed data for the Iowa Data Visualization project.

## Expected Data Files

### Raw CSV Files
Place the CSV files downloaded from Iowa.gov data here:

- `liquor-sales.csv` - Iowa liquor sales data
- `dui-charges.csv` - DUI charge records
- `sex-offenders.csv` - Sex offender registry data
- `amber-alerts.csv` - Amber alert records
- `incarceration.csv` - Incarceration data (optional)
- `school-proficiency.csv` - Math and reading proficiency data (optional)

### Processed Data
Processed and cleaned data files will be stored here:

- `processed/` - Directory for cleaned and transformed datasets

## Data Processing Notes

- Ensure all CSV files have proper headers
- Geographic data should include latitude and longitude fields
- Remove any sensitive personal information before committing
- Document any data transformations in this file
