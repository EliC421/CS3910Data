# Data Directory

This directory stores raw and processed datasets for county-level Iowa analysis.

## Current Raw Datasets (Present)

### Core crash + normalization inputs

- `Vehicle_Crashes_in_Iowa_20260312.csv`
  - Crash records with county, date, severity, injuries/fatalities, and point geometry.
- `County_Population_in_Iowa_by_Year_20260312.csv`
  - County population by year for rate normalization.
- `IowaCounties.geojson`
  - County boundaries and county attributes for mapping.

### Additional control/proxy datasets

- `Iowa_Probation_Recidivism_Status_20260312.csv`
  - Includes `Jurisdiction` and `Supervision Offense Subtype` (contains OWI values).
- `Iowa_Local_Area_Unemployment_Statistics.csv`
  - Includes county-level unemployment fields by month/year.
- `Iowa_Median_Household_Income_in_the_Past_12_Months_(ACS_5-Year_Estimates)_20260312.csv`
  - Includes county-level income values by year window.

### Liquor sales

- `Iowa_Liquor_Sales_20260312.csv`
  - Transaction-level liquor sales. Filtered to 2024+. Key fields: `County`, `Date`, `Sale (Dollars)`, `Volume Sold (Liters)`, `Bottles Sold`.

## Normalization Rule

- `ratePer100k = (countyTotal / countyPopulation) * 100000`

County join key normalization:

- trim spaces
- remove trailing `County`
- remove trailing `, Iowa`
- compare case-insensitively

## Data Notes

- Crash data provides a DUI proxy via `Drug or Alcohol` values not equal to `None Indicated`.
- Probation data provides OWI justice-system signal via `Supervision Offense Subtype = OWI`.
- If direct DUI charge counts are unavailable, document this proxy approach in analysis writeups.
- Liquor sales data is 2024+ only; `liquor_dollars_per_100k` will be blank for pre-2024 county-years.
